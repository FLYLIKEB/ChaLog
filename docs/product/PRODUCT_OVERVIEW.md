# 차멍 (ChaLog) 서비스 기획서

차를 마시는 순간을 기록하고, 새로운 차를 발견하는 모바일 앱.

---

## 프로젝트 개요

**차멍**은 차(茶)를 즐기는 사람들을 위한 모바일 우선 기록 및 커뮤니티 서비스입니다. 사용자는 마신 차를 평가하고 차록(茶錄)을 남기며, 다른 사람의 차록을 통해 새로운 차를 발견합니다. 혼자 마시는 순간부터 여럿이 함께하는 블라인드 테이스팅 세션까지, 차를 중심으로 한 모든 경험을 담을 수 있습니다.

---

## 핵심 가치 / 타겟 사용자

### 핵심 가치

- **기록**: 마신 차의 향미, 평점, 감상을 체계적으로 남긴다
- **발견**: 다른 사람의 공개 차록과 차담 게시판을 통해 새로운 차를 만난다
- **관리**: 내 찻장(보유 차 목록)을 관리하고 잔량을 추적한다
- **공유**: 시음 세션과 블라인드 테이스팅으로 함께하는 차 경험을 만든다

### 타겟 사용자

| 유형 | 설명 |
|------|------|
| 차 입문자 | 마셔본 차를 기록하며 취향을 파악하고 싶은 사람 |
| 차 애호가 | 체계적인 평가 기록과 찻장 관리가 필요한 사람 |
| 차 모임 참여자 | 블라인드 테이스팅 등 그룹 활동을 즐기는 사람 |
| 차 커뮤니티 이용자 | 차담 게시판에서 정보를 나누고 싶은 사람 |

## 주요 기능 목록

### 차록 (茶錄) — 차 기록

- 마신 차 검색 및 선택 (자동완성 지원)
- 5점 슬라이더 기반 항목별 평가 (향, 맛, 색, 여운 등)
- 자유 텍스트 감상 메모
- 공개/비공개 설정
- 차록 수정 및 삭제
- 다른 사람의 차록 저장(북마크)

### 차 탐색

- 홈 피드: 추천 차 및 공개 차록 목록
- 사색(차 검색): 이름 기반 자동완성 검색
- 차 상세 페이지: 차 정보 + 태그 + 관련 차록 모음
- 태그 기반 탐색

### 찻장 (茶藏) — 보유 차 관리

- 보유 중인 차 목록 관리
- 잔량 슬라이더로 남은 양 추적
- 찻장 아이템 추가 / 수정 / 삭제

### 시음 세션

- 개인 시음 세션 기록 (진행 중 → 요약)
- 세션 히스토리 조회

### 블라인드 테이스팅

- 세션 생성 및 초대 코드 발급
- 초대 코드로 참가
- 참가자별 평가 작성
- 세션 종료 후 결과 리포트 확인

### 차담 (茶談) — 커뮤니티 게시판

- 카테고리별 게시글 작성 (추천, 질문, 정보 등)
- 게시글 댓글 및 반응
- 게시글 수정 / 삭제

### 차집 (茶家) — 찻집 정보

- 찻집 상세 페이지 조회
- 찻집 등록 및 수정

### 사용자 기능

- 내 차록 모음 페이지
- 저장한 차록 모음
- 다른 사용자 프로필 조회
- 알림 수신
- 앱 설정

|------|--------|------|
| `/` | 홈 | 추천 차, 공개 차록 피드 |

| `/user/:id` | 사용자 프로필 | 다른 사용자의 공개 차록 |

## PWA 지원 사항

차멍은 Progressive Web App으로 제공됩니다. 별도 앱 설치 없이 모바일 홈 화면에 추가하여 네이티브 앱처럼 사용할 수 있습니다.

| 기능 | 지원 내용 |
|------|-----------|
| 홈 화면 추가 | iOS Safari, Android Chrome 모두 지원 |
| Standalone 모드 | 브라우저 UI 없이 전체 화면 앱 형태로 실행 |
| iOS 설치 안내 | 첫 방문 시 설치 방법 안내 배너 표시 |
| Service Worker | 정적 자산 캐싱으로 오프라인 환경에서도 기본 화면 유지 |
| 앱 아이콘 | 홈 화면에 표시되는 전용 아이콘 및 스플래시 이미지 |
