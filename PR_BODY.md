# [1단계 MVP] 피드 - 좋아요 및 북마크 기능

## 개요
노트에 좋아요와 북마크를 누를 수 있는 기능을 구현했습니다.

## 변경 사항

### 백엔드
#### 좋아요 기능
- ✅ NoteLike 엔티티 생성 (noteId, userId 복합 유니크 제약)
- ✅ 좋아요 테이블 생성 SQL 및 JS 스크립트 추가
- ✅ NotesService에 좋아요 토글, 조회 메서드 추가
- ✅ 노트 조회 시 좋아요 수 및 사용자 좋아요 여부 포함
- ✅ POST /notes/:id/like 엔드포인트 추가
- ✅ 중복 좋아요 방지 로직 구현
- ✅ E2E 테스트 추가 (11개 테스트 모두 통과)
- ✅ OptionalJwtAuthGuard 추가로 선택적 인증 지원

#### 북마크 기능
- ✅ NoteBookmark 엔티티 생성 (noteId, userId 복합 유니크 제약)
- ✅ 북마크 테이블 생성 SQL 및 JS 스크립트 추가
- ✅ NotesService에 북마크 토글 메서드 추가
- ✅ 노트 조회 시 북마크 여부 포함
- ✅ POST /notes/:id/bookmark 엔드포인트 추가
- ✅ 중복 북마크 방지 로직 구현

### 프론트엔드
#### 좋아요 기능
- ✅ Note 타입에 likeCount, isLiked 필드 추가
- ✅ notesApi.toggleLike 메서드 추가
- ✅ NoteCard에 좋아요 버튼 및 좋아요 수 표시
- ✅ NoteDetail에 좋아요 버튼 추가
- ✅ 좋아요 클릭 시 즉시 UI 업데이트
- ✅ NoteCard 레이아웃 수정: 이미지를 위에 배치하고 그 아래에 다른 요소 배치

#### 북마크 기능
- ✅ Note 타입에 isBookmarked 필드 추가
- ✅ notesApi.toggleBookmark 메서드 추가
- ✅ NoteCard에 북마크 버튼 추가
- ✅ NoteDetail에 북마크 버튼 추가
- ✅ 북마크 클릭 시 즉시 UI 업데이트
- ✅ 북마크한 노트는 북마크 아이콘으로 표시

## 테스트

- ✅ 예외 케이스 테스트 포함 (존재하지 않는 노트, 인증 없음 등)
